# UI Pack Review — 2026-05-16

## Verdict

The new UI haul is useful, but uneven.

- **Best pack for the current game:** `fantasy_ui_paid`
- **Best use right now:** selective decoration, not a total reskin
- **Best long-term homes:** shop, inventory, codex, main menu

## What fits the game well

### Strong fit

- dark ornate frames from `fantasy_ui_paid`
- lanterns, small flasks, potion bottles, and parchment props
- richer menu framing pieces
- future inventory/codex decorations

### Maybe, later

- wood signboards and rough boards
- banners
- backpacks, shields, books, crosses

These are good assets, but they pull the game more rustic/medieval. They should
be used where the game wants warmth, not across every UI surface.

### Weak fit right now

- full wood-panel replacement for the existing shop
- overusing the free mini packs
- replacing all current `pixel_ui` framing with the new brown UI wholesale

That would make the game feel stitched from two different menus.

## Pack-by-pack

| Pack | Read on it |
| --- | --- |
| `fantasy_ui_paid` | The valuable one. Lots of usable dark frames, plaques, icons, scrolls, and warm gold accents. |
| `fantasy_ui_free` | Cute sampler, but mostly redundant once the paid fantasy atlas is owned. |
| `medieval_ui_paid` | Great prop sheet. Better for accents than structure. |
| `medieval_ui_free` | Tiny sampler; useful but not a reason to change the game's look by itself. |

## Current recommendation

Use the new art in small, intentional doses:

1. Shop: title-area accents / merchant flavor
2. Main menu: future ornament pass if we want it warmer
3. Inventory: strongest future destination for the wood/parchment assets

This keeps the project visually coherent while still letting the new purchases
earn their keep.
