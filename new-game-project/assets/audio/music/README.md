# music/

Background music tracks. Added last — get SFX feeling great first.

Suggested structure once music production begins:
- combat_loop.ogg     ← main gameplay loop (loopable!)
- shop_loop.ogg       ← calm between-wave track
- main_menu.ogg       ← title screen music
- boss.ogg            ← optional boss encounter track

Use .ogg format for music in Godot — much smaller file size than .wav for long tracks.
All music should loop cleanly. Set loop points in Godot's import settings.
