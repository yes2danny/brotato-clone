# round_lose/

Defeat stings that play when the player dies or fails a run.

Examples of what belongs here:
- Sad trombone / descending tones
- Somber short musical stings
- Deflating / failure sounds

Should feel clearly "bad" but not annoying — players will hear this a lot.
Keep it short (1–3 seconds). A little dark humor tone fits the game's vibe.
