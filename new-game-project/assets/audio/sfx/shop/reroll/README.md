# shop/reroll/

Sounds for the shop reroll action — plays when the player spends gold to refresh the item selection.

What's here (play in this order):
1. dice_grab.wav    ← player picks up the dice
2. dice_shake_3.wav ← shaking / anticipation
3. dice_roll_3.wav  ← result landing

Play all three back-to-back in quick sequence for the full effect.
grab → shake → roll gives the reroll a satisfying little story before the new items appear.
