# enemies/death/

Sounds that play when an enemy is killed.

Examples of what belongs here:
- Generic death splat / pop
- Family-based variants later: small_death.wav, medium_death.wav, heavy_death.wav
- Special enemy deaths (boss, elite, treasure goblin) can go here too, named clearly

Start with one satisfying generic death sound. More variation = more juice, but keep
it realistic — one good sound beats ten mediocre ones.
