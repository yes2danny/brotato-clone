# weapons/

Sounds for weapon interactions — picking up, equipping, unequipping, and upgrading.

What's here:
- weapon_pick_up.wav      ← grabbing a weapon off the ground / from a drop
- weapon_equip_short.wav  ← equipping a weapon into a slot (short confirm sound)
- weapon_unequip.wav      ← removing / swapping out a weapon
- weapon_upgrade.wav      ← upgrading a weapon to the next tier (should feel rewarding!)

Subfolders:
- explosions/             ← explosive weapon impact sounds

Play weapon_upgrade.wav on level-up or star upgrade — make it feel like a big moment.
