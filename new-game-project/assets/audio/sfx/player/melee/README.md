# player/melee/

Sounds for melee weapon attacks — swings, swipes, punches, slashes.

Examples of what belongs here:
- Sword / knife swing (whoosh)
- Punch / fist impact
- Club or bat swipe

Separate from player/shoot/ because melee has a different feel — more physical, close-range.
Not implemented yet but the folder is ready when melee weapons are added to the roster.
