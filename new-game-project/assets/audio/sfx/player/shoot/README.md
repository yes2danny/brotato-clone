# player/shoot/

Sounds for the player firing weapons. One of the most-heard SFX in the game.

Examples of what belongs here:
- Generic bullet / projectile fire
- Per-weapon sounds if needed later (pistol, shotgun, SMG, etc.)

Start with one solid default fire sound. Per-weapon sounds can be added once the weapon
roster is stable. Prioritize this one — players hear it constantly.
