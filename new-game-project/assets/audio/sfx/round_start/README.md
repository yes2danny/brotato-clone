# round_start/

Sounds or stings that play when a new wave begins.

Examples of what belongs here:
- Tense build-up hit / impact
- Short dramatic sting ("here they come")
- Wave countdown beeps / alarm

Should feel like a signal to the player: enemies are incoming. Keep it punchy and clear.

## clock_ticking.wav — dual use
This one has two potential homes:
1. **Wave countdown** — loop it during the "wave starting in 3... 2... 1..." moment for tension
2. **Shop background loop** — play it quietly in the background while the shop is open,
   creating subtle pressure to hurry up and buy. Keep the volume low so it doesn't annoy.

Either way it should loop seamlessly. Check the loop point in Godot's import settings.
